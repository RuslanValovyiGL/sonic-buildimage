
# GLO.VE : CLI/API table

The config\_db to vtysh conversion in the table below for the most part relies on **sonic-frr-mgmt-framework**.

To use _**frr-mgmt-framework**_ add “frr\_mgmt\_framework\_config”: “true” to device DEVICE\_METADATA like this

{
    "DEVICE\_METADATA": {
        "localhost": {
            "frr\_mgmt\_framework\_config": "true",
        }
    }
}
